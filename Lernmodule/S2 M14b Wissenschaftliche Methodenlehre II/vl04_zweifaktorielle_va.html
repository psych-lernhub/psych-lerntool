M14b · VL04 Zweifaktorielle Varianzanalyse

Wissenschaftliche Methodenlehre II (Statistik 2) · Dirk Lubbe · Klausur Sept 2026 (WH) · Freitext/Rechnen

Lernfragen: 1. Warum zwei Faktoren? (Notation) · 2. Das zweifaktorielle Modell · 3. Die drei Hypothesen · 4. Quadratsummenzerlegung · 5. Freiheitsgrade & F-Tests · 6. Interaktion verstehen · 7. Teststärke & partielles η² · 8. Unbalancierte Pläne & QS-Typen
Vom ein- zum mehrfaktoriellen Design
Vorteile zweier Faktoren (Interaktion, weniger Fehler, Störvariablen); Notation p, q, n, N; Zell- und Randmittelwerte.
Phase 1 — Erst selbst überlegen

Eine Studie prüft die Wirkung dreier Medikationsstufen und nimmt zusätzlich das Geschlecht auf. Was gewinnt man durch den zweiten Faktor?

Drei Dinge: Man kann die Interaktion Medikation × Geschlecht prüfen, man bindet Varianz, die sonst im Fehler läge (→ mehr Teststärke), und man kann den Einfluss von Störvariablen reduzieren.
Phase 2 — Theoretischer Lernteil
Zwei Faktoren — warum?
Oft will man die Wirkung mehrerer Faktoren zugleich erfassen und den Effekt prüfen, der sich aus ihrer Kombination (Interaktion) ergibt. Zusätzlich sinkt der Fehleranteil, und Störvariablen lassen sich als zweiter Faktor kontrollieren.
Notation
Faktor A: p Stufen (a_i), Faktor B: q Stufen (b_j). Es gibt pq Zellen (Faktorstufenkombinationen), je n Personen → N = p·q·n. Einzelwert y_ijm (drei Indizes).
Zell- und Randmittelwerte
ĀB_ij = Zellmittel (eine Kombination). Ā_i, B̄_j = Randmittel (über den jeweils anderen Faktor gemittelt). Ḡ = Gesamtmittel.
Phase 3 — Interaktiv
Notation am Beispiel (3×2-Design, n = 5)
Schritt 1: Faktor A = Medikation (p = 3), Faktor B = Geschlecht (q = 2).
Schritt 2: Zellen = p·q = 3·2 = 6 Faktorstufenkombinationen.
Schritt 3: Gesamtzahl N = p·q·n = 3·2·5 = 30 Personen.
Phase 4 — Sofort-Abfrage
Das zweifaktorielle Modell
y_ijm = μ + α_i + β_j + αβ_ij + ε_ijm; Zerlegung der Abweichung vom Gesamtmittel.
Phase 1 — Erst selbst überlegen

Aus welchen vier Komponenten setzt sich im zweifaktoriellen Modell ein einzelner Messwert zusammen?

y_ijm = μ + α_i + β_j + αβ_ij + ε_ijm: Gesamtmittel + Haupteffekt A + Haupteffekt B + Interaktion + Fehler. Die Effekte addieren sich (nicht multiplizieren).
Phase 2 — Theoretischer Lernteil
Zerlegung der Beobachtung
(y_ijm − Ḡ) = (Ā_i − Ḡ) + (B̄_j − Ḡ) + (ĀB_ij − Ā_i − B̄_j + Ḡ) + (y_ijm − ĀB_ij) — also Haupteffekt A + Haupteffekt B + Interaktion + Fehler.
Populationsmodell
y_ijm = μ + α_i + β_j + αβ_ij + ε_ijm, mit ε ~ N(0, σ_e²). μ = Gesamtmittel, α_i = Haupteffekt A, β_j = Haupteffekt B, αβ_ij = Interaktion.
⚠ Verwechslung
Das Modell ist additiv. Ein „Produkt“ der Effekte (μ·α·β·…) ist falsch — αβ_ij ist ein eigener Effektterm, nicht das Produkt von α_i und β_j.
Phase 3 — Interaktiv
Welche Komponente ist das?
μ → das Gesamtmittel (Niveau aller Beobachtungen).
α_i → der Haupteffekt von Faktor A (Abweichung des A-Randmittels vom Gesamtmittel).
β_j → der Haupteffekt von Faktor B.
αβ_ij → die Interaktion; ε_ijm → der individuelle Fehler.
Phase 4 — Sofort-Abfrage
Die drei Nullhypothesen
Haupteffekt A (alle α_i = 0), Haupteffekt B (alle β_j = 0), Interaktion (alle αβ_ij = 0).
Phase 1 — Erst selbst überlegen

Wie viele Nullhypothesen prüft eine zweifaktorielle ANOVA — und welche?

Drei: Haupteffekt A (alle α_i = 0), Haupteffekt B (alle β_j = 0) und die Interaktion A×B (alle αβ_ij = 0).
Phase 2 — Theoretischer Lernteil
Haupteffekt A
H0: α_1 = α_2 = … = α_p = 0 — die Stufen von Faktor A unterscheiden sich (gemittelt über B) nicht.
Haupteffekt B
H0: β_1 = β_2 = … = β_q = 0 — die Stufen von Faktor B unterscheiden sich (gemittelt über A) nicht.
Interaktion A×B
H0: αβ_11 = … = αβ_pq = 0 (für alle i, j) — es gibt keine Wechselwirkung der beiden Faktoren.
Phase 3 — Interaktiv
Ordne zu: Haupteffekt A, Haupteffekt B oder Interaktion?
H0: α_1 = … = α_p = 0
H0: β_1 = … = β_q = 0
H0: αβ_11 = … = αβ_pq = 0
Der Effekt von Faktor A hängt von der Stufe von B ab
Effekt-Simulator — welche der drei Nullhypothesen?

Eine zweifaktorielle ANOVA prüft drei unabhängige Nullhypothesen: zwei Haupteffekte (Randmittel) und eine Interaktion (Zellenmuster).

Phase 4 — Sofort-Abfrage
Quadratsummenzerlegung
QS_tot = QS_A + QS_B + QS_AB + QS_e; QS_A = nq·Σ(Ā_i−Ḡ)², QS_B = np·Σ(B̄_j−Ḡ)², QS_AB.
Phase 1 — Erst selbst überlegen

In wie viele Anteile zerfällt die totale Quadratsumme bei zwei Faktoren — und warum mehr als beim einfaktoriellen Fall?

QS_tot = QS_A + QS_B + QS_AB + QS_e: zwei Haupteffekte, eine Interaktion und der Fehler. Einfaktoriell waren es nur QS_A + QS_e.
Phase 2 — Theoretischer Lernteil
Die Zerlegung
QS_tot = QS_A + QS_B + QS_AB + QS_e (additiv im balancierten Fall). Jeder Modell-Term (Haupteffekt A, Haupteffekt B, Interaktion, Fehler) erhält seine eigene Quadratsumme.
Die Formeln
QS_A = nq·Σ(Ā_i − Ḡ)², QS_B = np·Σ(B̄_j − Ḡ)², QS_AB = n·ΣΣ(ĀB_ij − Ā_i − B̄_j + Ḡ)², QS_e = ΣΣΣ(y_ijm − ĀB_ij)².
⚠ Verwechslung
QS_A wird mit nq gewichtet (Ā_i beruht auf q·n Personen), QS_B mit np. Nicht die Faktoren vertauschen: Faktor A bekommt q, Faktor B bekommt p.
Phase 3 — Interaktiv
Quadratsummenzerlegung Schritt für Schritt
Schritt 1: Gesamtvariabilität QS_tot = ΣΣΣ(y_ijm − Ḡ)².
Schritt 2: Haupteffekte: QS_A = nq·Σ(Ā_i − Ḡ)², QS_B = np·Σ(B̄_j − Ḡ)².
Schritt 3: Interaktion: QS_AB = n·ΣΣ(ĀB_ij − Ā_i − B̄_j + Ḡ)².
Schritt 4: Rest = Fehler QS_e. Kontrolle: QS_A + QS_B + QS_AB + QS_e = QS_tot.
Phase 4 — Sofort-Abfrage
Freiheitsgrade & die drei F-Tests
df_A=p−1, df_B=q−1, df_AB=(p−1)(q−1), df_e=pq(n−1); F_A/F_B/F_AB = MQ/MQ_e.
Phase 1 — Erst selbst überlegen

Wogegen werden Haupteffekt A, Haupteffekt B und die Interaktion jeweils getestet — und mit welchen Freiheitsgraden?

Alle drei werden gegen MQ_e geprüft: F = MQ_A/MQ_e, F = MQ_B/MQ_e, F = MQ_AB/MQ_e. df: p−1, q−1, (p−1)(q−1); Fehler pq(n−1).
Phase 2 — Theoretischer Lernteil
Freiheitsgrade
df_A = p−1, df_B = q−1, df_AB = (p−1)(q−1), df_e = pq(n−1), df_tot = N−1. Additiv: df_tot = df_A + df_B + df_AB + df_e.
Die drei F-Tests
MQ = QS/df. Alle Effekte werden gegen die Fehlervarianz geprüft: F_A = MQ_A/MQ_e, F_B = MQ_B/MQ_e, F_AB = MQ_AB/MQ_e. Jeweils Vergleich mit dem kritischen F-Wert.
⚠ Verwechslung
Im Nenner steht immer MQ_e — nicht MQ_AB oder eine Summe anderer MQ. Und es werden die mittleren Quadratsummen ins Verhältnis gesetzt, nicht die rohen QS.
Phase 3 — Interaktiv
Ordne die Freiheitsgrade zu: A, B, Interaktion oder Fehler?
df = p − 1
df = q − 1
df = (p − 1)(q − 1)
df = pq(n − 1)
Phase 4 — Sofort-Abfrage
Interaktion verstehen
Effekt von A hängt von Stufe B ab; ohne Interaktion parallele Linien/additiv; Interaktionskomponente ĀB_ij−Ā_i−B̄_j+Ḡ.
Phase 1 — Erst selbst überlegen

Im Mittelwert-Plot verlaufen die Linien für Männer und Frauen über die Dosisstufen genau parallel. Liegt eine Interaktion vor?

Nein. Parallele Linien bedeuten: der Effekt der Dosis ist für beide Geschlechter gleich → keine Interaktion. Die Zellmittel sind additiv (Ā_i + B̄_j − Ḡ).
Phase 2 — Theoretischer Lernteil
Was ist eine Interaktion?
Eine Interaktion bedeutet: der Effekt des einen Faktors hängt von der Stufe des anderen ab (z. B. die Differenz Männer/Frauen variiert über die Dosisstufen).
Ohne Interaktion = additiv
Ohne Interaktion sind die erwarteten Zellmittel Ā_i + B̄_j − Ḡ; im Plot zeigen sich parallele Linien. Die Interaktionskomponente lautet ĀB_ij − Ā_i − B̄_j + Ḡ.
⚠ Verwechslung
Eine Interaktion ist nicht dasselbe wie zwei vorhandene Haupteffekte. Haupteffekte und Interaktion sind unabhängig: beides kann einzeln oder zusammen auftreten.
Phase 3 — Interaktiv
Liegt eine Interaktion vor?
Die Linien im Mittelwert-Plot verlaufen parallel
Die Linien kreuzen sich
Die Differenz zwischen Männern und Frauen ist über alle Dosisstufen konstant
Der Effekt der Dosis ist bei Männern stärker als bei Frauen
Phase 4 — Sofort-Abfrage
Teststärke & partielles Eta-Quadrat
Zweiter Faktor bindet Varianz → kleineres MQ_e → größeres F; η²_p = QS_Effekt/(QS_Effekt+QS_e).
Phase 1 — Erst selbst überlegen

Dieselben Daten ergeben einfaktoriell F_A = 35,9, zweifaktoriell aber F_A = 74,5 — obwohl QS_A gleich bleibt. Wie kommt das?

Der zweite Faktor (Geschlecht) und die Interaktion binden Varianz, die einfaktoriell im Fehler steckte. Dadurch sinkt MQ_e (3,53 → 1,70) → der gleiche Zähler ergibt ein größeres F = mehr Teststärke.
Phase 2 — Theoretischer Lernteil
Mehr Teststärke durch den 2. Faktor
Ein relevanter zweiter Faktor (plus Interaktion) zieht Variabilität aus dem Fehler. QS_A bleibt gleich, aber MQ_e sinkt → F steigt. Im Beispiel klettert F_A von 35,9 auf 74,5.
Partielles Eta-Quadrat
Effektmaß mehrfaktorieller ANOVAs: η²_p = QS_Effekt / (QS_Effekt + QS_e) — der Anteil aufgeklärter Variabilität des Effekts relativ zu der nicht durch andere Faktoren aufgeklärten Varianz.
⚠ Verwechslung
η²_p bezieht sich auf QS_Effekt + QS_e (nicht QS_tot). Signifikanz (F) und Effektstärke (η²_p) sind getrennt zu berichten.
Phase 3 — Interaktiv
Warum steigt F_A? Schritt für Schritt
Schritt 1: Einfaktoriell landet die Geschlechtsvarianz im Fehler: QS_e = 95,3, df 27, MQ_e = 3,53.
Schritt 2: Zweifaktoriell binden B und A×B diese Varianz: QS_e = 40,8, df 24, MQ_e = 1,70.
Schritt 3: QS_A = 253,4 bleibt gleich, MQ_A = 126,7 bleibt gleich.
Schritt 4: F_A = 126,7/1,70 ≈ 74,5 statt 126,7/3,53 ≈ 35,9 → mehr Teststärke.
Phase 4 — Sofort-Abfrage
Unbalancierte Pläne & Quadratsummen-Typen
Ungleiche n → nicht orthogonal; gewichtete vs. ungewichtete Mittel; Typ I (gewichtet, reihenfolgeabhängig, additiv) vs. Typ III (ungewichtet).
Phase 1 — Erst selbst überlegen

Die Zellen Ihres 2×2-Plans enthalten unterschiedlich viele Personen. Warum ist die Auswertung jetzt heikler — und welche „Typen“ von Quadratsummen gibt es?

Bei ungleichen n sind die Faktoren nicht mehr orthogonal; Randmittel kann man gewichtet oder ungewichtet bilden. Typ III testet die ungewichteten, Typ I die gewichteten Mittel (reihenfolgeabhängig, einziger additiver Typ).
Phase 2 — Theoretischer Lernteil
Unbalancierter Plan
Unbalanciert = ungleiche Zellbesetzung (ungleiche n). Dann sind die Faktoren nicht mehr orthogonal — die Quadratsummen „überlappen“, und die Randmittel können gewichtet (nach Zellgröße) oder ungewichtet berechnet werden.
Quadratsummen-Typen
Typ III → ungewichtete Mittel (übliche Standardauswertung). Typ I → gewichtete Mittel; die Haupteffekt-Hypothesen hängen von der Reihenfolge der Faktoren ab.
Additivität & balancierter Fall
Unbalanciert ist nur Typ I additiv (QS_tot = ΣQS); Typ II/III/IV nicht. Bei balancierten Plänen liefern dagegen alle Typen identische Ergebnisse.
Phase 3 — Interaktiv
Typ I vs. Typ III — Schritt für Schritt
Schritt 1: Zellen ungleich besetzt → Faktoren nicht orthogonal → QS-Typ wird relevant.
Schritt 2: Typ III prüft die ungewichteten Randmittel — unabhängig von der Faktor-Reihenfolge.
Schritt 3: Typ I prüft die gewichteten Mittel und hängt von der Reihenfolge (zuerst A oder zuerst B) ab.
Schritt 4: Nur Typ I ist additiv; bei balancierten Plänen sind alle Typen gleich.
