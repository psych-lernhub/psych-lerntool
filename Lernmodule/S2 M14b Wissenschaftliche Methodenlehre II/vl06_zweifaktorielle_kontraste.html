Vier Arten von Kontrasten
Treatmentkontraste & einfache Haupteffekte (Haupteffekte); Interaktionskontraste & einfache Treatmentkontraste (Interaktion).
Phase 1 — Erst selbst überlegen

Nach einem signifikanten zweifaktoriellen Test will man genauer hinschauen. Welche vier Kontrastarten stehen zur Verfügung — und wofür?

Für Haupteffekte: Treatmentkontraste und einfache Haupteffekte. Für Interaktionen: Interaktionskontraste und einfache Treatmentkontraste.
Phase 2 — Theoretischer Lernteil
Zwei für Haupteffekte
Treatmentkontraste (analog zur einfaktoriellen ANOVA) und einfache Haupteffekte dienen der genaueren Analyse von Haupteffekten.
Zwei für Interaktionen
Interaktionskontraste und einfache Treatmentkontraste dienen der genaueren Analyse von Interaktionen.
Datenbeispiel (Boik 1979)
3×4-Plan: A = Therapie (Kontrolle / Verhaltenstherapie / Gesprächstherapie), B = Medikation (Placebo / D1 / D2 / D3), n = 6, AV = edr (Hautreaktion bei Blutphobie).

Kontrastarten-Simulator (zweifaktoriell)

Im zweifaktoriellen Plan gibt es vier Kontrastarten — sie unterscheiden sich darin, ob sie über die Stufen des anderen Faktors gemittelt oder auf eine Stufe bedingt sind.

Phase 4 — Sofort-Abfrage
Treatmentkontraste
Kontrast der Randmittel eines Faktors (analog einfaktoriell); D(A)=Σc_i·Ā_i, QS = nq·D²/Σc_i², 1 df.
Phase 1 — Erst selbst überlegen

Warum steht in der Quadratsumme eines Treatmentkontrasts der Faktor nq und nicht nur n?

Weil ein Treatmentkontrast über die Randmittel Ā_i gebildet wird, und jedes Randmittel auf q·n Personen beruht (alle B-Stufen zusammengefasst). Daher: QS = nq·D(A)²/Σc_i².
Phase 2 — Theoretischer Lernteil
Was ist ein Treatmentkontrast?
Möchte man spezifische Stufen eines Faktors vergleichen, nutzt man Treatmentkontraste — identisch zum Kontrast der einfaktoriellen ANOVA. Die anderen Faktoren werden dabei ignoriert.
Berechnung
D(A) = Σc_i·Ā_i (mit Σc_i = 0); QS = nq·D(A)²/Σc_i²; F = QS/MQ_e; 1 df.
Beispiel
Kontrolle vs. beide Therapien, c = (1, −½, −½): D = 47,9 − 0,5·33,9 − 0,5·38,5 = 11,7; QS = 6·4·11,7²/1,5 = 2190,24; F = 2190,24/19,15 ≈ 114,4.
Phase 3 — Interaktiv
Rechenweg Treatmentkontrast
Schritt 1: Koeffizienten festlegen (Σc_i = 0), z. B. c = (1, −½, −½).
Schritt 2: Kontrastwert D(A) = Σc_i·Ā_i = 11,7.
Schritt 3: QS = nq·D²/Σc² = 6·4·11,7²/1,5 = 2190,24.
Schritt 4: F = QS/MQ_e = 2190,24/19,15 ≈ 114,4 (1 df).
Phase 4 — Sofort-Abfrage
Einfache Haupteffekte (simple main effects)
A-Stufen unter einer B-Stufe; QS_{A|bj} = n·Σ_i(ĀB_ij − B̄_j)², df = p − 1.
Phase 1 — Erst selbst überlegen

Was ist ein „einfacher Haupteffekt" — und wie unterscheidet er sich vom gewöhnlichen Haupteffekt?

Ein einfacher Haupteffekt prüft die Unterschiedlichkeit der A-Stufen unter einer einzelnen Stufe von B — wie eine einfaktorielle ANOVA für A, beschränkt auf die Beobachtungen einer B-Stufe. Der gewöhnliche Haupteffekt mittelt dagegen über alle B-Stufen.
Phase 2 — Theoretischer Lernteil
Definition
Einfache Haupteffekte beziehen sich auf die Unterschiedlichkeit der Stufen von A unter den einzelnen Stufen von B — vergleichbar mit einer einfaktoriellen ANOVA für A, nur für die Daten einer B-Stufe.
Berechnung
QS_{A|bj} = n·Σ_i(ĀB_ij − B̄_j)², mit df = p − 1. F = [QS_{A|bj}/(p − 1)] / MQ_e.
Beispiel (A unter b1)
QS = 6·[(50,2−48,6)²+(49,9−48,6)²+(45,7−48,6)²] = 75,96; F = (75,96/2)/19,15 ≈ 1,98. Analog b2: 8,24, b3: 24,05, b4: 65,48.
Phase 3 — Interaktiv
Rechenweg einfacher Haupteffekt (A unter b1)
Schritt 1: Zellmittel unter b1: 50,2 / 49,9 / 45,7; Randmittel B̄_1 = 48,6.
Schritt 2: QS = 6·[(50,2−48,6)²+(49,9−48,6)²+(45,7−48,6)²] = 75,96.
Schritt 3: MQ = QS/(p−1) = 75,96/2 = 37,98.
Schritt 4: F = 37,98/19,15 ≈ 1,98 (df = p−1 = 2).
Phase 4 — Sofort-Abfrage
Interaktionskontraste
Spezifische 2×2-Interaktion; D = ΣΣc_ij·ĀB_ij, QS = n·D²/ΣΣc², 1 df.
Phase 1 — Erst selbst überlegen

Wie testet man gezielt EINE bestimmte Wechselwirkung, etwa (Verhaltens- vs. Gesprächstherapie) × (Placebo vs. hohe Dosis)?

Mit einem Interaktionskontrast — einer spezifischen 2×2-Interaktion. Sein Wert ist D = ΣΣc_ij·ĀB_ij, wobei die Koeffizienten das Produkt der beiden Einzelkontraste sind: z. B. (a2−a3)×(b1−b4).
Phase 2 — Theoretischer Lernteil
Was ist ein Interaktionskontrast?
Mit Interaktionskontrasten untersucht man die Interaktion mehrstufiger Faktoren anhand spezifischer 2×2-Interaktionen (zwei Stufen von A gegen zwei Stufen von B).
Berechnung
D = ΣΣc_ij·ĀB_ij; QS = n·D²/ΣΣc_ij²; F = QS/MQ_e; 1 df. Die 2×2-Koeffizienten sind das Produkt der beiden Einzelkontraste.
Beispiel (a2−a3)×(b1−b4)
D = ĀB_21 − ĀB_31 − ĀB_24 + ĀB_34 = 49,9 − 45,7 − 19,0 + 32,7 = 17,9; QS = 6·17,9²/4 = 480,6; F = 480,6/19,15 ≈ 25,1.
Phase 3 — Interaktiv
Vorzeichen der 2×2-Koeffizienten bestimmen
Schritt 1: Einzelkontrast A: (a2 − a3) → a2 bekommt +, a3 bekommt −.
Schritt 2: Einzelkontrast B: (b1 − b4) → b1 bekommt +, b4 bekommt −.
Schritt 3: Zellkoeffizient = Produkt: ĀB_21 (+·+ = +), ĀB_31 (−·+ = −), ĀB_24 (+·− = −), ĀB_34 (−·− = +).
Schritt 4: D = +ĀB_21 − ĀB_31 − ĀB_24 + ĀB_34 = 17,9.
Phase 4 — Sofort-Abfrage
Einfache Treatmentkontraste
Kontrast der A-Zellmittel innerhalb einer B-Stufe; D(A|bj) = Σc_i·ĀB_ij, QS = n·D²/Σc², 1 df.
Phase 1 — Erst selbst überlegen

Ein einfacher Treatmentkontrast und ein einfacher Haupteffekt arbeiten beide „innerhalb einer B-Stufe". Was unterscheidet sie?

Der einfache Haupteffekt vergleicht alle p A-Stufen unter b_j (df = p−1). Der einfache Treatmentkontrast ist ein gerichteter Einzelvergleich daraus (df = 1): D(A|bj) = Σc_i·ĀB_ij.
Phase 2 — Theoretischer Lernteil
Definition
Ein einfacher Treatmentkontrast für A ist ein Kontrast von Zellmittelwerten, die alle zur gleichen Stufe von B gehören (z. B. die A-Stufen nur unter b1).
Berechnung
D(A|bj) = Σc_i·ĀB_ij; QS = n·D(A|bj)²/Σc_i²; F = QS/MQ_e; 1 df.
Beispiel
D_1(A|b1) = 2·50,2 − 49,9 − 45,7 = 4,8; QS = 6·4,8²/6 = 23,04; F = 23,04/19,15 ≈ 1,20. Analog b2: 16,36, b3: 38,07, b4: 101,56.
Phase 3 — Interaktiv
Rechenweg einfacher Treatmentkontrast (b1)
Schritt 1: Koeffizienten c = (2, −1, −1) für die A-Zellmittel unter b1.
Schritt 2: D(A|b1) = 2·50,2 − 49,9 − 45,7 = 4,8.
Schritt 3: QS = n·D²/Σc² = 6·4,8²/6 = 23,04.
Schritt 4: F = 23,04/19,15 ≈ 1,20 (1 df).
Phase 4 — Sofort-Abfrage
Signifikanzprüfung & Freiheitsgrade
Alle vier gegen MQ_e; drei mit 1 df, einfacher Haupteffekt mit p − 1.
Phase 1 — Erst selbst überlegen

Wogegen werden die vier Kontrastarten geprüft — und welche hat als Einzige nicht genau einen Freiheitsgrad?

Alle vier werden an MQ_e relativiert. Drei sind gerichtete Einzelvergleiche mit 1 df; nur der einfache Haupteffekt vergleicht p Gruppen → df = p − 1.
Phase 2 — Theoretischer Lernteil
Gemeinsamer Nenner: MQ_e
Jede Kontrast-Quadratsumme wird an der Fehlervarianz MQ_e relativiert: F = QS/MQ_e (bzw. [QS/(p−1)]/MQ_e beim einfachen Haupteffekt).
Freiheitsgrade
Treatmentkontrast, Interaktionskontrast und einfacher Treatmentkontrast sind gerichtete Einzelvergleiche → 1 df. Der einfache Haupteffekt vergleicht p Gruppen → p − 1 df.

Phase 4 — Sofort-Abfrage
Gewichtung (nq vs. n) & Rechenbeispiel
Randmittel → nq, Zellmittel → n; Interaktionskontrast D = 17,9 → F = 25,1.
Phase 1 — Erst selbst überlegen

Welcher Gewichtungsfaktor (nq oder n) steht in der Quadratsumme der jeweiligen Kontrastart?

Der Treatmentkontrast nutzt nq (er beruht auf Randmitteln mit q·n Personen). Alle drei anderen Arten beruhen auf Zellmitteln (n Personen) und nutzen daher n.
Phase 2 — Theoretischer Lernteil
Randmittel → nq
Ein Randmittel Ā_i fasst alle q Stufen des anderen Faktors zusammen und beruht auf q·n Personen. Deshalb wird die QS eines Treatmentkontrasts mit nq gewichtet.
Zellmittel → n
Ein Zellmittel ĀB_ij beruht auf nur n Personen. Einfacher Haupteffekt, Interaktionskontrast und einfacher Treatmentkontrast nutzen daher den Faktor n.
⚠ Verwechslung
Nur der Treatmentkontrast nutzt nq. Wer beim Interaktionskontrast versehentlich nq statt n einsetzt, erhält eine q-fach zu große Quadratsumme.
Phase 3 — Interaktiv
Gewicht in der QS: nq oder n?
Treatmentkontrast QS_D(A)
Einfacher Haupteffekt QS_{A|bj}
Interaktionskontrast QS_D
Einfacher Treatmentkontrast QS_D(A|bj)
Phase 4 — Sofort-Abfrage
A-posteriori-Korrekturen (Scheffé-artig)
S-Werte je Kontrastart: Interaktionskontrast (p−1)(q−1)·F; einfacher Treatmentkontrast (df_A+df_AB)·F.
Phase 1 — Erst selbst überlegen

Hat man die Kontraste erst nach dem Datenblick gewählt (a posteriori) — wie sichert man das Signifikanzniveau ab?

Über einen Scheffé-artigen kritischen S-Wert, der das Niveau über alle möglichen Kontraste hält. Für Interaktionskontraste: S = (p−1)(q−1)·F_krit (im Beispiel 6·3,12 = 18,72).
Phase 2 — Theoretischer Lernteil
Warum korrigieren?
A-posteriori-Vergleiche (nach dem Datenblick gewählt) müssen über alle möglichen Kontraste abgesichert werden — analog zum Scheffé-Verfahren der einfaktoriellen ANOVA. Der empirische F-Wert muss den kritischen S-Wert erreichen.
S-Wert für Interaktionskontraste
S = (p − 1)(q − 1) · F_{(p−1)(q−1), df_e; 1−α}. Beispiel: 6·F_{6,60;0,99} = 6·3,12 = 18,72.
Andere Kontrastarten
Einfacher Haupteffekt: S = (p−1)q·F/(p−1). Einfacher Treatmentkontrast: S = (df_A + df_AB)·F = (2+6)·2,82 = 22,56. Treatmentkontrast (Haupteffekt): S = (p−1)·F.
Phase 3 — Interaktiv
Rechenweg Scheffé-S für einen Interaktionskontrast (3×4-Plan)
Schritt 1: Freiheitsgrade der Interaktion: df_AB = (p−1)(q−1) = 2·3 = 6.
Schritt 2: kritischer F-Wert: F_{6,60;0,99} = 3,12.
Schritt 3: S = df_AB·F = 6·3,12 = 18,72.
Schritt 4: Der Interaktionskontrast ist signifikant, wenn sein empirischer F-Wert > 18,72.
