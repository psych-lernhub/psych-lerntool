Scheinkorrelation & Kausalität
Korrelation ≠ Kausalität; Scheinkorrelation = Zusammenhang von x0 und x1 wird durch eine Drittvariable x2 verursacht.
Phase 1 — Erst selbst überlegen

In einer Studie korrelieren die Anzahl der Polizist*innen und die Anzahl krimineller Delikte einer Kommune deutlich positiv. Heißt das: mehr Polizei verursacht mehr Kriminalität?

Nein. Beides wächst mit der Kommunengröße (Einwohnerzahl). Hält man die Einwohnerzahl konstant, verschwindet der paradoxe Zusammenhang — es ist eine Scheinkorrelation. Eine Korrelation beweist nie Kausalität.
Phase 2 — Theoretischer Lernteil
Korrelation
Maß für die Enge des linearen Zusammenhangs zweier Variablen: rxy = Sxy/(Sx·Sy). Sind beide Variablen dichotom → Phi-Koeffizient; ist eine dichotom und eine metrisch → punkt-biseriale Korrelation.
Korrelation ≠ Kausalität
Ein deutlicher Zusammenhang verleitet dazu, eine Ursache-Wirkungs-Beziehung zu unterstellen. Aber: eine vorhandene Korrelation kann die Kausalität nicht nachweisen — sie muss durch inhaltliche Überlegungen begründet werden.
Scheinkorrelation
x0 und x1 korrelieren nur, weil eine Drittvariable x2 beide beeinflusst. Beispiel: x2 = Kommunengröße treibt sowohl die Zahl der Delikte (x0) als auch die Polizeistärke (x1) nach oben.
Phase 3 — Interaktiv
So entsteht die Scheinkorrelation (Schritt für Schritt)
Dorf, Kleinstadt und Großstadt unterscheiden sich in der Einwohnerzahl x2.
Mit x2 steigen beide: die Zahl der Delikte x0 ↑ und die Zahl der Polizist*innen x1 ↑.
→ x0 und x1 korrelieren positiv, obwohl kein direkter (kausaler) Zusammenhang besteht. Bei konstant gehaltener Einwohnerzahl hätte sich der eigentlich plausible (negative) Zusammenhang gezeigt.
Phase 4 — Sofort-Abfrage
Variablen konstant halten / kontrollieren
Kontrollieren = konstant halten = Einfluss einer Variable eliminieren/bereinigen; statistisch statt experimentell.
Phase 1 — Erst selbst überlegen

Was ist gemeint, wenn man sagt, eine Variable werde „kontrolliert" oder „konstant gehalten"?

Ihr Einfluss wird aus den anderen Variablen eliminiert / bereinigt. Geht das nicht durch direktes Eingreifen (Experiment), kontrolliert man die Variable statistisch — z. B. per partieller Korrelation.
Phase 2 — Theoretischer Lernteil
Drei Wörter, ein Vorgang
Herauspartialisieren = konstant halten = kontrollieren. Der Einfluss einer Variable wird aus einer oder mehreren anderen Variablen „eliminiert" bzw. „bereinigt".
Experimentell vs. statistisch
Im Experiment hält man eine Störvariable durch direktes Eingreifen konstant. Ist das nicht möglich (z. B. das Alter der Kinder), bleibt nur die statistische Kontrolle.
⚠ Der naive Weg und sein Preis
Man könnte die Korrelation nur in einer altershomogenen Teilstichprobe (z. B. nur 8-Jährige) berechnen. Nachteil: der Stichprobenumfang sinkt drastisch → oft nicht praktikabel.
Phase 3 — Interaktiv
Zwei Wege, eine Variable konstant zu halten
Experimentell: direktes Eingreifen — z. B. von vornherein nur gleichaltrige Kinder testen.
Statistisch: rechnerisch bereinigen über die partielle Korrelation — geht auch nachträglich, ohne Stichprobenverlust.
Beide Wege zielen auf dasselbe: den Einfluss der Drittvariable ausschalten.
Phase 4 — Sofort-Abfrage
Beispiel & bivariate Korrelation
n=15 Kinder; r01 = S01/(S0·S1) = 0,89; Verdacht auf Drittvariable Alter (x2).
Phase 1 — Erst selbst überlegen

n = 15 Kinder: Abstraktionsfähigkeit (x0) und sensomotorische Koordination (x1) korrelieren mit r01 = 0,89. Warum sollte man hier misstrauisch werden?

Weil das Alter (x2) beide Fähigkeiten beeinflusst. Ein Teil des hohen Zusammenhangs könnte allein auf die Drittvariable Alter zurückgehen — eine (teilweise) Scheinkorrelation.
Phase 2 — Theoretischer Lernteil
Die Daten
n = 15 Kinder, drei Merkmale: x0 = Abstraktionsfähigkeit, x1 = sensomotorische Koordination, x2 = Alter. Vermutung: x2 beeinflusst sowohl x0 als auch x1.
Kennwerte
Standardabweichungen s0 = 2,282 und s1 = 2,484; Kovarianz s01 = 5,057.
Bivariate Korrelation
r01 = S01/(S0·S1) = 5,057/(2,282·2,484) = 0,89 — sehr hoch. Verdacht auf die Drittvariable Alter.
Phase 3 — Interaktiv
Rechenweg r01 (Schritt für Schritt)
Standardabweichungen: s0 = 2,282, s1 = 2,484.
Kovarianz: s01 = 5,057.
r01 = 5,057 / (2,282 · 2,484) = 5,057 / 5,668 = 0,89.
Phase 4 — Sofort-Abfrage
Statistische Kontrolle per Regression (Herauspartialisieren)
x0 und x1 auf x2 regredieren, Residuen x0*/x1* bilden; ihre Korrelation = partielle Korrelation r01·2 = 0,72.
Phase 1 — Erst selbst überlegen

Wie bekommt man eine Korrelation zwischen x0 und x1, die vom Alter x2 unbeeinflusst ist?

Man sagt x0 und x1 jeweils per Regression aus x2 vorher und behält nur die Residuen (x0*, x1*) — die vom Alter bereinigten Anteile. Deren Korrelation ist die partielle Korrelation r01·2.
Phase 2 — Theoretischer Lernteil
Partielle Korrelation (Definition)
r01·2 = die vom Einfluss von x2 bereinigte Korrelation zwischen x0 und x1 — gewonnen mit Hilfe der Regressionsrechnung.
Grafische Notation
Kästchen = beobachtete Variable · Oval = nicht beobachtet (z. B. Residuum) · Doppelpfeil = Korrelation · gerichteter Pfeil = Regression (AV ← UV).
Vorgehen (Herauspartialisieren)
I. x0 (bzw. x1) aus x2 vorhersagen. II. Residuen x0* = x0 − x̂0 bilden (= Herauspartialisieren). Die Korrelation der Residuen ist r01·2.
Ergebnis im Beispiel
x̂0 = 0,464 + 1,246·x2 und x̂1 = −1,130 + 1,420·x2 → r01·2 = r(x0*, x1*) = 0,72 (von 0,89 gesunken).
Phase 3 — Interaktiv
Herauspartialisieren in 4 Schritten
x0 auf x2 regredieren: x̂0 = 0,464 + 1,246·x2.
x1 auf x2 regredieren: x̂1 = −1,130 + 1,420·x2.
Residuen bilden: x0* = x0 − x̂0 und x1* = x1 − x̂1 (die vom Alter bereinigten Werte).
Residuen korrelieren: r01·2 = r(x0*, x1*) = 0,72.
Phase 4 — Sofort-Abfrage
Rechnerische Durchführung (Formel)
r01·2 = (r01 − r02·r12)/(√(1−r02²)·√(1−r12²)); r01·2 = 0 ⇔ r01 = r02·r12.
Phase 1 — Erst selbst überlegen

Muss man für die partielle Korrelation immer erst alle Residuen einzeln ausrechnen?

Nein. Es genügen die drei bivariaten Korrelationen: r01·2 = (r01 − r02·r12) / (√(1−r02²)·√(1−r12²)).
Phase 2 — Theoretischer Lernteil
Formel (1. Ordnung)
r01·2 = (r01 − r02·r12) / (√(1−r02²)·√(1−r12²)). Es werden nur die drei Roh-Korrelationen benötigt — kein Umweg über die einzelnen Residuen.
Beispiel
r01=0,89, r02=0,77, r12=0,80 → (0,89 − 0,616)/(0,638·0,600) = 0,274/0,383 = 0,72 — dasselbe Ergebnis wie der Residuenweg.
⚠ Wann ändert sich nichts — wann alles?
Zähler = 0, also r01 = r02·r12 → r01·2 = 0. Ist x2 mit x0 oder x1 unkorreliert (r02=0 oder r12=0) → r01·2 = r01 (es gibt nichts zu bereinigen).
Phase 3 — Interaktiv
Wie wirkt x2? Vergleiche r01·2 mit der bivariaten r01.
x2 beeinflusst x0 und x1 gleichsinnig positiv (Confounder).
r02 = 0 (x2 ist mit x0 unkorreliert).
r01 = r02·r12 (der Zusammenhang ist ganz durch x2 erklärbar).
r12 = 0 (x2 ist mit x1 unkorreliert).
Phase 4 — Sofort-Abfrage
Signifikanztest (Fisher-Z)
Fisher-Z von r01·2 und ρ0; z = √(n−3−(k−2))·(Z−Z0) ~ N(0,1); 1. Ordnung √(n−4); |z|>1,96 (5%) / >2,58 (1%).
Phase 1 — Erst selbst überlegen

Die partielle Korrelation beträgt 0,72. Ist sie signifikant von null verschieden?

Per Fisher-Z-Test: Z(0,72) = 0,908, Z0 = 0; z = √(n−3−(k−2))·(Z−Z0) = √11·0,908 = 3,01 > 2,58 → signifikant auf dem 1%-Niveau.
Phase 2 — Theoretischer Lernteil
Schritt 1 — Fisher-Z
r01·2 und der Vergleichswert ρ0 werden in Fisher-Z transformiert: Z (für r01·2) und Z0 (für ρ0).
Prüfgröße
z = √(n − 3 − (k−2)) · (Z − Z0) ~ N(0,1) — nur bei multivariater Normalverteilung. k = Anzahl beteiligter Variablen, (k−2) = Anzahl kontrollierter Variablen; 1. Ordnung (k=3) → √(n−4).
Entscheidung (zweiseitig)
|z| > 1,96 → signifikant auf 5%-Niveau. |z| > 2,58 → signifikant auf 1%-Niveau.
Phase 3 — Interaktiv
z-Wert berechnen (Beispiel)
Fisher-Z von r01·2 = 0,72: Z = 0,908. Für ρ0 = 0: Z0 = 0.
Freiheitsgrad-Wurzel: √(n−3−(k−2)) = √(15−3−1) = √11 ≈ 3,317.
z = 3,317 · (0,908 − 0) = 3,01.
|3,01| > 2,58 → signifikant auf dem 1%-Niveau.
Phase 4 — Sofort-Abfrage
Multivariate Normalverteilung (Voraussetzung)
Test nötig, aber kein ausgereifter vorhanden; univariate Normalität notwendig, nicht hinreichend; Histogramme prüfen.
Phase 1 — Erst selbst überlegen

Der Signifikanztest verlangt eine multivariate Normalverteilung. Wie prüft man die?

Es gibt keinen ausgereiften Test. Faustregel: alle Variablen müssen univariat normalverteilt sein (Histogramme: symmetrisch, unimodal) — das ist aber notwendig, nicht hinreichend.
Phase 2 — Theoretischer Lernteil
Voraussetzung
Die Prüfgröße folgt nur bei multivariater Normalverteilung der beteiligten Variablen approximativ einer Standardnormalverteilung.
Prüfung
Kein ausgereifter Test vorhanden → Histogramme aller Variablen ansehen: symmetrisch und unimodal?
⚠ Notwendig ≠ hinreichend
Multivariate Normalität ⇒ alle univariaten normalverteilt. Die Umkehrung gilt nicht: trotz univariater Normalität aller Variablen kann die multivariate Annahme verletzt sein.
Phase 3 — Interaktiv
Normalverteilung prüfen — Faustregel
Jede Variable einzeln: Histogramm — symmetrisch und unimodal?
Ist nicht jede Variable univariat normal → multivariate Normalität ausgeschlossen.
Sind alle univariat normal → multivariate Normalität wahrscheinlich, aber nicht garantiert (notwendig, nicht hinreichend).
Phase 4 — Sofort-Abfrage
Höhere Ordnungen & semipartielle Korrelation
Ordnung = Anzahl kontrollierter Variablen; semipartiell = nur EINE Variable bereinigt (Nenner ohne √(1−r02²)).
Phase 1 — Erst selbst überlegen

Was ändert sich, wenn man (a) zwei statt einer Drittvariable kontrolliert oder (b) nur eine der beiden Variablen bereinigt?

(a) → partielle Korrelation 2. Ordnung r01·23. (b) → semipartielle (Teil-)Korrelation sr. Bei sr fehlt im Nenner der Faktor √(1−r02²), weil x0 unbereinigt bleibt.
Phase 2 — Theoretischer Lernteil
Ordnung = Anzahl kontrollierter Variablen
1. Ordnung: r01·2 (eine Variable). 2. Ordnung: r01·23 = (r01·2 − r03·2·r13·2)/(√(1−r03·2²)·√(1−r13·2²)) — sie baut nur auf partiellen Korrelationen 1. Ordnung auf.
Semipartielle (Teil-)Korrelation
Nur EINE Variable wird bereinigt. sr1 = (r01 − r02·r12)/√(1−r12²) (nur x1 um x2 bereinigt). Gegenüber der partiellen Korrelation fehlt im Nenner der Faktor √(1−r02²).
Phase 3 — Interaktiv
Partielle oder semipartielle Korrelation?
Beide zu korrelierenden Variablen (x0 und x1) werden um x2 bereinigt.
Nur x1 wird um x2 bereinigt; x0 bleibt unbereinigt.
Nenner der Formel: √(1−r02²)·√(1−r12²).
Nenner der Formel: nur √(1−r12²).
Partielle Korrelation 1. oder 2. Ordnung?
r01·2 — eine Variable (x2) wird kontrolliert.
r01·23 — zwei Variablen (x2 und x3) werden kontrolliert.
Die Korrelationen, die in r01·23 eingehen (r01·2, r03·2, r13·2).
Aus dem Zusammenhang von x0 und x1 werden gleichzeitig zwei Variablen herauspartialisiert.
Korrelations-Simulator — was wird herauspartialisiert?

Schlüssel: Bei der partiellen Korrelation wird Z aus beiden Variablen herauspartialisiert, bei der semipartiellen nur aus einer.

Phase 4 — Sofort-Abfrage
Häufige Verwechslungen
Die fünf klausurkritischen Gegensatzpaare dieser VL sicher trennen.

Scheinkorrelation (unkontrolliert)

x0 und x1 korrelieren nur über die Drittvariable x2

im Beispiel r01 = 0,89 (hoch)

Drittvariable Alter nicht herausgerechnet

vs.

Partielle Korrelation (x2 kontrolliert)

x0 und x1 um x2 bereinigt

im Beispiel r01·2 = 0,72 (gesunken)

von der Drittvariable unbeeinflusst

⚠ Fehlaussage: Die partielle Korrelation r01·2 enthält den Einfluss der Drittvariable x2 noch vollständig, während die bivariate Korrelation r01 ihn bereits herausrechnet.

Partielle Korrelation r01·2

BEIDE Variablen werden um x2 bereinigt

Nenner: √(1−r02²)·√(1−r12²)

= Korrelation der Residuen x0* und x1*

vs.

Semipartielle Korrelation sr

nur EINE Variable wird bereinigt

Nenner: nur √(1−r12²)

z. B. nur x1 um x2 bereinigt

⚠ Fehlaussage: Bei der semipartiellen Korrelation werden beide zu korrelierenden Variablen um die Drittvariable bereinigt.

Partielle Korrelation 1. Ordnung

EINE Variable kontrolliert (x2)

r01·2

geht aus den bivariaten Korrelationen hervor

vs.

Partielle Korrelation 2. Ordnung

ZWEI Variablen kontrolliert (x2, x3)

r01·23

geht aus Korrelationen 1. Ordnung hervor

⚠ Fehlaussage: In eine partielle Korrelation 2. Ordnung gehen ausschließlich bivariate Rohkorrelationen ein.

Korrelation

Maß der Enge des linearen Zusammenhangs

symmetrisch (r01 = r10)

beweist KEINE Kausalität

vs.

Kausalität

gerichtete Ursache-Wirkungs-Beziehung

muss inhaltlich begründet werden

nicht aus einer Korrelation ableitbar

⚠ Fehlaussage: Eine hinreichend hohe Korrelation zwischen zwei Variablen weist deren kausalen Zusammenhang nach.

Univariate Normalverteilung

jede Variable für sich normalverteilt

per Histogramm beurteilbar (symmetrisch/unimodal)

notwendig für multivariate Normalität

vs.

Multivariate Normalverteilung

Voraussetzung des Signifikanztests

kein ausgereifter Test vorhanden

univariate Normalität nicht hinreichend

⚠ Fehlaussage: Wenn alle Variablen einzeln (univariat) normalverteilt sind, ist damit automatisch auch die multivariate Normalverteilung gesichert.
