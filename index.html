🧠 Psychologie — Lernmodule

Interaktive Lernmodule, Übungssets und Modell-Trainer zur Klausurvorbereitung. Dein Fortschritt wird lokal in deinem Browser gespeichert.

📦 Fortschritt sichern & auf ein anderes Gerät übertragen
Was ist das? Dein Lernfortschritt wird nur in dem Browser/Gerät gespeichert, in dem du lernst — er wandert nicht von allein auf andere Geräte. Mit einem Code kannst du ihn mitnehmen (z. B. Handy → Laptop) oder als Sicherung aufbewahren.
So geht's:
  1. Auf dem Gerät, wo du gelernt hast: auf ⬇️ Code erzeugen drücken, dann 📋 Kopieren. Schick dir den Code an dich selbst (z. B. per WhatsApp, E-Mail oder in eine Notiz).
  2. Auf dem anderen Gerät: diese Seite öffnen, den Code unten in das Feld einfügen und auf ⬆️ Fortschritt aus Code laden drücken.
  3. Die Seite lädt kurz neu — oben bei 📊 Karten gelernt siehst du sofort, dass dein Stand übernommen wurde. ✓

💡 Der Code ist eine Momentaufnahme: lernst du danach weiter, erzeuge vor dem nächsten Übertragen einfach einen neuen Code. „Laden" überschreibt den Stand der enthaltenen Module auf diesem Gerät.
